	<div class="title">
		<h2>在Mac OS X Lion下安装Ruby on Rails</h2>	
	</div>
	
	<div class="content"><p>自败了Mac Pro后，才开始着手Ruby on Rails开发。之前写了一篇《<a title="Ruby on Rails开发入门" href="http://www.hoowolf.net/2011/09/24/ruby-on-rails-development/" target="_blank">Ruby on Rails开发入门</a>》，但这个是在Windows 7开发环境下，网上很多大牛也说过，Windows开发环境下进行Rails开发，可能会遇到一些很诡异的问题，确实是这样，自己在实践一些测试工具如cucumber等时候，就很纠结。</p>
<h3>安装环境要求</h3>
<p>Mac OS X Lion环境下，正式安装前，需要确认一些安装所需的必要工具：<span id="more-706"></span></p>
<p>1. Xcode开发工具    Xcode是苹果推出的开发工具，它是免费的，可以在Mac Store中下载。如果你已经安装了，那么可以打开，在菜单栏中选择Xcode -&gt; Preference -&gt; Download，然后点击Command Line Tools对应的install即可。如果你没有安装的话，也可以直接去下载<a href="https://developer.apple.com/downloads/index.action?version=3.2" target="_blank">Command Line Tools for Xcode</a>。其实我推荐先安装Xcode，因为后一种方法也得注册Apple Developer ID才能进去，既然如此还不如一气呵成呢。</p>
<p>之所以需要安装Xcode，是因为其中包含了一些安装Rails所需要的编译包，如果你没有安装Command Line Tools的话，在利用RVM安装Ruby 1.9.3的时候，会报出如下错误：</p>
<pre class="brush:shell">rvm requires autoreconf to install the selected ruby interpreter however autoreconf was not found in the PATH.</pre>
<p>原因是缺少autoreconf和automake包。这两个包具体作用，有兴趣的可以深究一下。</p>
<p>2. 如果安装了Xcode，那么git就已经安装了，可以确认一下：</p>
<pre class="brush:shell">$ git --version
git version 1.7.7.5 (Apple Git-26)</pre>
<p>这些都确认以后，就可以开始安装<a href="https://rvm.beginrescueend.com/rvm/install/" target="_blank">RVM（Ruby Version Manager）</a>了。RVM是学习Rails的人必须了解的一样工具。因为Mac本身自带了Ruby，但版本比较老，而我们开发所需要的一般都是最新的Ruby。由于Mac相关环境可能会用到Ruby的旧版本，直接删除旧版本会影响系统相关特性，所以RVM可以很灵活地帮我们管理不同版本。</p>
<h3>安装RVM：</h3>
<pre class="brush:shell">$ bash -s stable &lt; &lt;(curl -s https://raw.github.com/wayneeseguin/rvm/master/binscripts/rvm-installer)
#或者
$ bash &lt; &lt;( curl -s https://rvm.beginrescueend.com/install/rvm )</pre>
<p>接着需要配置RVM快捷命令，否则你无法在终端里直接使用rvm命令：</p>
<pre class="brush:shell">$ echo '[[ -s "$HOME/.rvm/scripts/rvm" ]] &amp;&amp; . "$HOME/.rvm/scripts/rvm" ' &gt;&gt; ~/.bash_profile</pre>
<p>很明显，意思就是直接将脚本插入到了.bash_profile中。运行：</p>
<pre class="brush:shell">$ source ~/.bash_profile</pre>
<p>或者是直接重启Terminal，然后可以输入如下命令进行验证：</p>
<pre class="brush:shell">$ type rvm | head -1
rvm is a function</pre>
<p>这样即证明rvm命令配置成功了。有兴趣的还可以运行</p>
<pre class="brush:shell">$ echo $PATH</pre>
<p>试一下，也能验证。</p>
<blockquote><p>如果有人喜欢用Sublime Text 2作为编辑器进行相关操作的话，首先也需要在$PATH路径中配置subl为快捷命令，具体如下：</p>
<pre class="brush:shell">$ ln -s /Applications/Sublime\ Text\ 2.app/Contents/SharedSupport/bin/subl /usr/bin/subl</pre>
<p>这样的话，上述配置rvm命令也可以这样操作：</p>
<pre class="brush:shell">$ subl ~/.bash_profile</pre>
<p>然后在编辑文本里添加如下即可：</p>
<pre class="brush:shell">[[ -s "$HOME/.rvm/scripts/rvm" ]] &amp;&amp; . "$HOME/.rvm/scripts/rvm"</pre>
</blockquote>
<h3>Ruby安装</h3>
<p>下面就可以安装ruby 1.9.3版本了：</p>
<pre class="brush:shell">$ rvm install 1.9.3
#安装完成后进行设置
$ rvm use 1.9.3  #或者
$ rvm --default 1.9.3
#验证设置
$ruby -v
ruby 1.9.3p125 (2012-02-16 revision 34643) [x86_64-darwin11.3.0]</pre>
<p>现在确认Ruby Gems，Gems是Ruby的包管理器（Package Manager）。</p>
<pre class="brush:shell">$ gem -v
1.8.21
#如果不是最新版本，可以
$ gem update --system</pre>
<h3>Database</h3>
<p>作为初学者，可以选择SQLite3：他已经在OS X系统环境里了。</p>
<p>确认SQLite3的版本：</p>
<pre class="brush:shell">$ sqlite3 -version
3.7.7 2011-06-25 16:35:41</pre>
<p>当然也可以使用gem安装一下：</p>
<pre class="brush:shell">$ gem install sqlite3</pre>
<h3>安装Rails</h3>
<p>铺垫很久了，终于到了最后：</p>
<pre class="brush:shell">$ gem install rails
#验证版本
$ rails -v
3.2.2</pre>
<p>这样环境就搭建成功了，可以开始Rails开发学习了。</p>
<blockquote><p>配置过程中，还折腾了brew，brew就相当于Linux下的apt-get，具体可以看<a href="http://mxcl.github.com/homebrew/" target="_blank">Homebrew项目主页</a>相关介绍。</p></blockquote>
<p><span style="color: #ff0000;">2013-03-10更新：</span>ruby 2.0.0已经发布了，rvm install 2.0.0即可安装。如果安装过程中出现各种问题，建议参考「<a href="http://www.interworks.com/blogs/ckaukis/2013/03/05/installing-ruby-200-rvm-and-homebrew-mac-os-x-108-mountain-lion" target="_blank"><em>Installing Ruby 2.0.0 with RVM and Homebrew on Mac OS X 10.8 Mountain Lion</em></a>」这篇文章，图文都说的很清楚~</p>
<p>参考资料：</p>
<p>1. <a href="http://tecparatodos.com/2011/07/24/installing-ruby-on-rails-on-mac-os-x-lion/" target="_blank">Installing Ruby on Rails on Mac OS X Lion</a></p>
<p>2. <a href="http://www.vpsee.com/2011/04/install-ruby-and-rails-on-mac-os-x/" target="_blank">在 Mac OS X Snow Leopard 上安装 Ruby 和 Rails</a></p>
<p>3. <a href="http://ruby.railstutorial.org/ruby-on-rails-tutorial-book?version=3.2" target="_blank">Ruby on Rails Tutorial</a></p>
<p>4. <a href="http://www.leonzhang.com/2012/02/22/launch_sublime_text_2_from_mac_terminal/" target="_blank">在Mac的Terminal中使用Sublime Text 2</a></p>
